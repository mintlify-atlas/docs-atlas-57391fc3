## Overview

A DataFrame is a two-dimensional labeled data structure with columns of potentially different types. It is similar to a spreadsheet or SQL table, combining:

- **Rows** (observations) identified by an index
- **Columns** (variables) identified by column names  
- **Data** stored in a columnar format for efficient access

## Constructor

<ParamField path="data" type="DataFrameData" required>
  Object mapping column names to arrays of values. All arrays must have the same length.
  
  ```ts
  { name: ['Alice', 'Bob'], age: [25, 30] }
  ```
</ParamField>

<ParamField path="options" type="DataFrameOptions">
  Configuration options for DataFrame construction.
  
  <Expandable title="properties">
    <ParamField path="index" type="(string | number)[]">
      Custom row labels (defaults to 0, 1, 2, ...)
    </ParamField>
    
    <ParamField path="columns" type="string[]">
      Custom column order (defaults to Object.keys order)
    </ParamField>
    
    <ParamField path="copy" type="boolean" default="true">
      Whether to copy data on construction. Set to false for performance if data ownership can be transferred.
    </ParamField>
  </Expandable>
</ParamField>

```ts
import { DataFrame } from 'deepbox/dataframe';

const df = new DataFrame({
  name: ['Alice', 'Bob', 'Charlie'],
  age: [25, 30, 35],
  score: [85.5, 92.0, 78.5]
});

console.log(df.shape);  // [3, 3]
console.log(df.columns);  // ['name', 'age', 'score']
```

## Data Access

### get()

Get a column as a Series.

<ParamField path="column" type="string" required>
  Column name to retrieve
</ParamField>

<ParamField path="guard" type="(value: unknown) => value is T">
  Optional type guard function to validate and type-narrow column values
</ParamField>

<ResponseField name="return" type="Series<T>">
  Series containing the column data
</ResponseField>

```ts
const df = new DataFrame({ age: [25, 30, 35], name: ['Alice', 'Bob', 'Carol'] });
const ageSeries = df.get('age');  // Series([25, 30, 35])

// With type guard
const ages = df.get('age', (v): v is number => typeof v === 'number');
```

### loc()

Access a row by label (label-based indexing).

<ParamField path="row" type="string | number" required>
  The index label of the row
</ParamField>

<ResponseField name="return" type="Record<string, unknown>">
  Object mapping column names to values for that row
</ResponseField>

```ts
const df = new DataFrame(
  { age: [25, 30], name: ['Alice', 'Bob'] },
  { index: ['row1', 'row2'] }
);
df.loc('row1');  // { age: 25, name: 'Alice' }
```

### iloc()

Access a row by integer position (position-based indexing).

<ParamField path="position" type="number" required>
  The integer position (0-based)
</ParamField>

<ResponseField name="return" type="Record<string, unknown>">
  Object mapping column names to values for that row
</ResponseField>

```ts
const df = new DataFrame({ age: [25, 30], name: ['Alice', 'Bob'] });
df.iloc(0);  // { age: 25, name: 'Alice' }
df.iloc(1);  // { age: 30, name: 'Bob' }
```

const df = new DataFrame({ a: [1, 2, 3, 4, 5], b: [6, 7, 8, 9, 10] });

### tail()

<ParamField path="n" type="number" default="5">
  Number of rows to return
</ParamField>

<ResponseField name="return" type="DataFrame">
  New DataFrame with last n rows

### filter()

Filter rows based on a boolean predicate function.

<ParamField path="predicate" type="(row: Record<string, any>) => boolean" required>
  Function that returns true for rows to keep
</ParamField>

<ResponseField name="return" type="DataFrame">
  New DataFrame with filtered rows
</ResponseField>

<ParamField path="columns" type="string[]" required>

```ts

### drop()

<ParamField path="columns" type="string[]" required>
  Array of column names to drop

<ResponseField name="return" type="DataFrame">

</ResponseField>

const df = new DataFrame({ a: [1, 2], b: [3, 4], c: [5, 6] });

```

## Sorting & Grouping

### sort()

Sort DataFrame by one or more columns.

<ParamField path="by" type="string | string[]" required>
  Column name or array of column names to sort by
</ParamField>

<ParamField path="ascending" type="boolean" default="true">
  Sort in ascending order
</ParamField>

<ResponseField name="return" type="DataFrame">
  New sorted DataFrame
</ResponseField>

```ts
const df = new DataFrame({ age: [30, 25, 35], name: ['Bob', 'Alice', 'Carol'] });
df.sort('age');  // Sorted by age ascending
df.sort(['age'], false);  // Sorted by age descending
```

### groupBy()

Group DataFrame by one or more columns.

<ParamField path="by" type="string | string[]" required>
  Column name or array of column names to group by
</ParamField>

<ResponseField name="return" type="DataFrameGroupBy">
  DataFrameGroupBy object for aggregation operations
</ResponseField>

```ts
const df = new DataFrame({
  category: ['A', 'B', 'A', 'B'],
  value: [10, 20, 30, 40]
});
const grouped = df.groupBy('category');
grouped.sum();  // Sum values by category
```

## Joining & Combining

### join()

Join with another DataFrame using SQL-style join.

Uses hash join algorithm for O(n + m) time complexity.

<ParamField path="other" type="DataFrame" required>
  DataFrame to join with
</ParamField>

<ParamField path="on" type="string" required>
  Column name to join on (must exist in both DataFrames)
</ParamField>

<ParamField path="how" type="'inner' | 'left' | 'right' | 'outer'" default="'inner'">
  Type of join operation:
  - `inner`: Only rows with matching keys in both DataFrames
  - `left`: All rows from left, matched rows from right (nulls for non-matches)
  - `right`: All rows from right, matched rows from left (nulls for non-matches)
  - `outer`: All rows from both DataFrames (nulls for non-matches)
</ParamField>

<ResponseField name="return" type="DataFrame">
  New DataFrame with joined data
</ResponseField>

```ts
const customers = new DataFrame({
  id: [1, 2, 3],
  name: ['Alice', 'Bob', 'Charlie']
});
const orders = new DataFrame({
  id: [1, 1, 2, 4],
  product: ['Laptop', 'Mouse', 'Keyboard', 'Monitor']
});

// Inner join - only customers with orders
const inner = customers.join(orders, 'id', 'inner');

// Left join - all customers, with/without orders
const left = customers.join(orders, 'id', 'left');
```

### merge()

Merge with another DataFrame using SQL-style merge.

More flexible than join() - supports different column names for join keys.

<ParamField path="other" type="DataFrame" required>
  DataFrame to merge with
</ParamField>

<ParamField path="options" type="MergeOptions" required>
  Merge configuration options.
  
  <Expandable title="properties">
    <ParamField path="on" type="string">
      Column name to join on (must exist in both DataFrames)
    </ParamField>
    
    <ParamField path="left_on" type="string">
      Column name in left DataFrame
    </ParamField>
    
    <ParamField path="right_on" type="string">
      Column name in right DataFrame
    </ParamField>
    
    <ParamField path="how" type="'inner' | 'left' | 'right' | 'outer'" default="'inner'">
      Join type
    </ParamField>
    
    <ParamField path="suffixes" type="[string, string]" default="['_x', '_y']">
      Suffix for duplicate column names
    </ParamField>
  </Expandable>
</ParamField>

<ResponseField name="return" type="DataFrame">
  New DataFrame with merged data
</ResponseField>

```ts
const employees = new DataFrame({
  emp_id: [1, 2, 3],
  name: ['Alice', 'Bob', 'Charlie']
});
const salaries = new DataFrame({
  employee_id: [1, 2, 4],
  salary: [50000, 60000, 55000]
});

const result = employees.merge(salaries, {
  left_on: 'emp_id',
  right_on: 'employee_id',
  how: 'left'
});
```

### concat()

Concatenate with another DataFrame.

<ParamField path="other" type="DataFrame" required>
  DataFrame to concatenate
</ParamField>

<ParamField path="axis" type="Axis" default="0">
  Axis to concatenate along:
  - 0 or "rows" or "index": Stack vertically (append rows)
  - 1 or "columns": Stack horizontally (append columns)
</ParamField>

<ResponseField name="return" type="DataFrame">
  Concatenated DataFrame
</ResponseField>

```ts
const df1 = new DataFrame({ a: [1, 2], b: [3, 4] });
const df2 = new DataFrame({ a: [5, 6], b: [7, 8] });
df1.concat(df2, "rows");  // Stack vertically: 4 rows
df1.concat(df2, "columns");  // Stack horizontally: 4 columns
```

## Missing Data

### fillna()

Fill missing values (null or undefined) with a specified value.

<ParamField path="value" type="unknown" required>
  Value to use for filling missing values
</ParamField>

<ResponseField name="return" type="DataFrame">
  New DataFrame with missing values filled
</ResponseField>

```ts
const df = new DataFrame({ a: [1, null, 3], b: [4, 5, undefined] });
df.fillna(0);  // Replace null/undefined with 0
```

### dropna()

Drop rows that contain any missing values (null or undefined).

<ResponseField name="return" type="DataFrame">
  New DataFrame with rows containing missing values removed
</ResponseField>

```ts
const df = new DataFrame({ a: [1, null, 3], b: [4, 5, 6] });
df.dropna();  // Only keeps rows 0 and 2
```

### isnull()

<ResponseField name="return" type="DataFrame">
  DataFrame of booleans (true = null/undefined, false = not null)
</ResponseField>

### notnull()

Return boolean DataFrame showing non-null values.

<ResponseField name="return" type="DataFrame">
  DataFrame of booleans (true = not null, false = null/undefined)
</ResponseField>

## Transformation

### apply()

Apply a function along an axis of the DataFrame.

<ParamField path="fn" type="(series: Series<unknown>) => Series<U>" required>
  Function to apply to each Series
</ParamField>

<ParamField path="axis" type="Axis" default="0">
  Axis to apply along (0=columns, 1=rows)
</ParamField>

<ResponseField name="return" type="DataFrame">
  New DataFrame with function applied
</ResponseField>

```ts
const df = new DataFrame({ a: [1, 2, 3], b: [4, 5, 6] });
df.apply(series => series.map(x => Number(x) * 2), 0);
```

### rename()

Rename columns or index labels.

<ParamField path="mapper" type="Record<string, string> | ((name: string) => string)" required>
  Object mapping old names to new names, or function to transform names
</ParamField>

<ParamField path="axis" type="0 | 1" default="1">
  0 for index, 1 for columns
</ParamField>

<ResponseField name="return" type="DataFrame">
  New DataFrame with renamed labels
</ResponseField>

```ts
const df = new DataFrame({ a: [1, 2], b: [3, 4] });
df.rename({ a: 'x', b: 'y' }, 1);  // Rename columns a->x, b->y
df.rename((name) => name.toUpperCase(), 1);  // Uppercase all column names
```

### reset_index()

Reset index to default integer index.

<ParamField path="drop" type="boolean" default="false">
  If true, don't add old index as column
</ParamField>

<ResponseField name="return" type="DataFrame">
  New DataFrame with reset index
</ResponseField>

```ts
const df = new DataFrame({ a: [1, 2] }, { index: ['x', 'y'] });
df.reset_index();  // Index becomes [0, 1], adds 'index' column
```

### set_index()

Set a column as the index.

<ParamField path="column" type="string" required>
  Column name to use as index
</ParamField>

<ParamField path="drop" type="boolean" default="true">
  If true, remove the column after setting it as index
</ParamField>

<ResponseField name="return" type="DataFrame">
